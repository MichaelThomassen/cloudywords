About Cloudy Words

Cloudy Words is designed and developed by Michael Thomassen as a fun introduction to Angular—his first time using the framework.

All words in the game were generated by ChatGPT and have not been proofread in any way, shape, or form (yet) — so expect a few surprises!

The game is intended for mobile devices in portrait mode but works reasonably well on desktop browsers too.

The source code is available on GitHub

We use Google Analytics to count page visits and see where our users come from (anonymized). No personal data is stored.

Have fun!
